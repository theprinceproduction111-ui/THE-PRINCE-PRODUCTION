## 🚀 STEP-BY-STEP: GitHub + Vercel Deployment

### STEP 1 — GitHub Account
1. Go to **https://github.com** → Sign Up (if not already)
2. Click your profile icon → **"Your repositories"**
3. Click green **"New"** button
4. Repository name: `the-prince-production`
5. Set to **Public**
6. Click **"Create repository"**

### STEP 2 — Upload Files to GitHub
1. On your new repo page, click **"uploading an existing file"**
2. **Drag & drop** these 3 files:
   - `index.html`
   - `admin.html`
   - `vercel.json`
3. Scroll down → Click **"Commit changes"**
4. ✅ Files are now on GitHub!

### STEP 3 — Deploy on Vercel
1. Go to **https://vercel.com** → Sign up with your GitHub account
2. Click **"Add New Project"**
3. Click **"Import"** next to `the-prince-production`
4. Framework Preset: **Other** (leave default)
5. Click **"Deploy"** 🚀
6. Wait 30 seconds...
7. ✅ **YOUR WEBSITE IS LIVE!**

### STEP 4 — Your Live URLs
After deployment Vercel gives you a URL like:
- **Website:** `https://the-prince-production.vercel.app`
- **Admin Panel:** `https://the-prince-production.vercel.app/admin`
